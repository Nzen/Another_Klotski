Instructions

  • wasd keys move just the cursor around the board.
  • shift + WASD keys will drag the shape under the cursor, provided there's room.
  • Moving the red block to the diagonally hatched area solves the puzzle.
  • Save game with ~ ; restore game with %. Game state in/out of the box below. (Push key with box deselected.)
